# Hit&Blow
TypeScriptの練習用プログラムです。

実際のゲーム例:
https://www.p-game.jp/game243/

## 環境構築
Node.jsのインストール
https://nodejs.org/ja/download/

確認コマンド
```
node --version
```
```
npm --version
```

TSのインストール
```
npm install -g typescript@4.3.5
```

もしソースコード内のprocessモジュールで怒られている場合
```
npm install -D @types/node@16.4.13
```




## 実行の仕方
TSのコンパイル
```
npm run build
```

CLIでの実行
```
npm run start
```
